# 禁用浏览器visibilitychange事件

> [!WARNING]
> 1. 此脚本使用DeepSeek生成
> 2. 仅供学习，请勿用于违法违规用途

此脚本用于彻底禁用页面中的visibilitychange事件，从而阻止网页中的JavaScript代码检测标签页的切换、窗口的失焦或聚焦等用户行为。如果你不希望网页检测你的这些行为，这个脚本会很有用。

## 使用方法

1. 在浏览器中安装Tampermonkey插件
2. 打开Tampermonkey菜单，选择添加新脚本
3. 在弹出的页面中，粘贴main.js中的完整代码
4. 将`@match`这一行的网址替换为你希望生效的目标网站，或者用`*://*/*`来表示所有网址（不推荐）
5. 点击File > Save保存
6. 到目标网站上刷新页面，观察Tampermonkey插件图标上的数字是否增加

## 注意事项
1. 代码由DeepSeek生成，经过测试后可以正常运行
2. 代码仅供学习和测试用途，请勿用于违法违规用途
3. 检查是否切换标签页的方法不止visibilitychange事件一种（但这是最简单、最稳定且最直接的一种，被广泛采用），部分网站需要具体情况具体分析
